📖 项目介绍

Free Notes 是一个轻量级的浏览器扩展，专为需要随时记录想法的用户设计。它会在每个网页的书签栏下方显示一个可收纳的便签卡片，支持多分类管理、导入导出等功能。

✨ 主要功能

📝 快速记录

在任何网页点击箭头按钮即可打开便签，支持新建、编辑、删除便签

🏷️ 分类管理

支持多分类管理，右击分类可重命名、换色、删除，让便签更有条理

📤 数据迁移

支持导入导出功能，便于数据备份和迁移到其他设备

🎨 简洁界面

简洁美观的界面设计，不会干扰正常的网页浏览体验

🔒 隐私保护

数据本地存储，不会上传到任何服务器，保护您的隐私

⚡ 轻量高效

占用资源极少，不会影响网页加载速度和浏览器性能

🚀 安装说明

Chrome 浏览器

  1. 下载扩展文件包
  2. 打开 Chrome 浏览器，进入扩展管理页面 (chrome://extensions/)
  3. 开启"开发者模式"
  4. 点击"加载已解压的扩展程序"，选择扩展文件夹
  5. 安装完成后，在书签栏下方即可看到便签按钮

Edge 浏览器

  1. 下载扩展文件包
  2. 打开 Edge 浏览器，进入扩展管理页面 (edge://extensions/)
  3. 开启"开发人员模式"
  4. 点击"加载解压缩的扩展"，选择扩展文件夹
  5. 安装完成后即可使用

📋 使用说明

基本操作

  • 打开便签：点击书签栏下方的箭头按钮
  • 新建便签：点击"+"按钮创建新便签
  • 编辑便签：点击便签内容区域进行编辑
  • 删除便签：点击便签右上角的删除按钮
  • 收起便签：点击箭头按钮收起便签面板

分类管理

  • 创建分类：在分类下拉菜单中选择"新建分类"
  • 重命名分类：右击分类名称选择"重命名"
  • 更改颜色：右击分类选择"更改颜色"
  • 删除分类：右击分类选择"删除分类"

数据管理

  • 导出数据：在设置页面点击"导出"按钮，下载 JSON 文件
  • 导入数据：在设置页面点击"导入"按钮，选择之前导出的 JSON 文件
  • 清除数据：在设置页面点击"清除所有数据"按钮（谨慎操作）

🔧 技术架构

技术栈

Manifest V3 JavaScript HTML5 CSS3 Chrome Extension API Local Storage

文件结构

manifest.json # 扩展配置文件 background.js # 后台脚本 content.js # 内容脚本 popup.html # 弹出窗口 popup.js # 弹出窗口脚本 popup.css # 弹出窗口样式 options.html # 设置页面 note.css # 便签样式 icon16.png # 16x16 图标 icon48.png # 48x48 图标 icon128.png # 128x128 图标 operation-demo.png # 操作示例图

核心功能实现

  • 内容脚本 (content.js)：负责在网页中注入便签界面和交互逻辑
  • 后台脚本 (background.js)：处理扩展的生命周期和数据存储
  • 弹出窗口 (popup.html/js)：提供快速访问和基本设置功能
  • 设置页面 (options.html)：提供详细的使用指南和配置选项

⚠️ 注意事项

  • 便签在扩展设置页和浏览器导航页不可见，这是正常的安全限制
  • 数据存储在浏览器本地，清除浏览器数据会导致便签丢失
  • 建议定期导出数据作为备份
  • 扩展需要访问所有网址的权限才能在所有网页显示便签
  • 如果遇到问题，请尝试重新安装扩展
